# 🍳 My First Agent Project: Персональный Шеф

Устали от очередного бесконечного выбора блюда после тяжелого рабочего дня? Хочется, чтобы хоть с одной проблемой из вашей жизни разобрался кто-то другой? 

Представляю вам оригинальное решение вопроса вашей вечерней трапезы: уникальный проект на базе **LangChain** и **LangGraph**, который использует возможности современных LLM для планирования вашего ужина.

---

## 🔥 Используемые технологии

*   🐍 **Язык исполнения** — Python 3.12+
*   🧠 **LangChain / LangGraph** — «Мозг» агента и логика переходов
*   🌐 **Tavily Search** — Инструмент точечного поиска рецептов в интернете
*   🤖 **OpenAI / GoogleGenAI** — Языковые модели (планирование и распознавание картинок)
*   ✅ **Pydantic** — Строгая валидация данных

---

## 🚀 Запуск!

### 1. Склонируйте репозиторий 
```bash
git clone https://github.com/thezeroxx/langchain_path
cd имя-репозитория
```

### 2. Настройка окружения 🛠
```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Установка зависимостей 📦
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения 🔑
Создайте файл `.env` в корневой папке и добавьте туда свои ключи (используйте `.env.example` как шаблон):
```text
TAVILY_API_KEY=your_key
OPENROUTER_API_KEY=your_key
GOOGLE_API_KEY=your_key
LANGCHAIN_API_KEY=your_key
```

## 📂 Структура проекта

*   📄 `chef.py` — основной файл запуска (входная точка).
*   ⚙️ `models.py` — инициализация и настройка моделей.
*   🛠 `utils.py` — вспомогательные функции.
*   📌 `constants.py` — константы и системные настройки.
