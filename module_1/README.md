# My First Agent Project
*Персональный Шеф*

Устали от очередного бесконечного выбора блюда после тяжелого рабочего дня?
Хочется, чтобы хоть с одной проблемой из вашей жизни разобрался кто-то другой?
Тогда представляю вам оригинальное решение вопроса вашей вечерней трапезы:

Уникальный Проект на базе Langchain и Langgraph, который использует LLM от OpenAI для решения задач 
с помощью инструментов Поиска (Tavily) и Распознавания Картинки (LLM от GoogleGenAI)

### Используемые Технологии
    ### Язык исполнения -- Python 3.12+
    ### **LangChain / LangGraph** -- "Мозг" Агента
    ### **Tavily Search** -- Инструмент поиска в Интернете
    ### **LLM от OpenAI / GoogleGenAI** -- Языковые Модели
    ### **Pydantic** -- Валидация Данных

## Запуск! ##

### 1. Склонируйте репозиторий 

    `git clone https://github.com
    cd имя-репозитория`


### 2. Настройка окружения

    `python -m venv .venv
    .venv\Scripts\activate`


### 3. Установка библиотек и зависимостей

    `pip install -r requirements.txt`


### 4. Настройте переменные окружения
    Создайте файл `.env` в корневой папке и добавьте туда свои ключи (используйте `.env.example` как шаблон):
    
    `TAVILY_API_KEY=your_key
    OPENROUTER_API_KEY=your_key
    GOOGLE_API_KEY=your_key
    LANGCHAIN_API_KEY=your_key`

# Структура проекта
    * `chef.py` — основной файл запуска (входная точка).
    * `models.py` — инициализация моделей.
    * `utils.py` — вспомогательные функции.
    * `constants.py` — константы и настройки.
